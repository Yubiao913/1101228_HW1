# 🏠 個人房貸試算器 (Mortgage Calculator)

採用 C# Windows Forms 架構開發，提供使用者直觀、便捷的個人房貸試算工具。

---

## 🚀 核心功能特色 (Features)

**彈性參數輸入**：支援房屋總價、自備款、年利率、貸款年限及寬限期等輸入設定 。
**自備款切換**：提供單選鈕（RadioButton）讓使用者自由切換以「百分比 (%)」或「固定金額 (元)」輸入自備款。
**寬限期邏輯運算**：區分寬限期內（只繳利息）與寬限期後（本息平均攤還）的每月應繳金額 。

---

## 🛠️ 使用說明書 (User Guide)

1. **設定房屋總價**：於「房屋總價」欄位輸入預期的購屋總金額 。
2. **選擇自備款模式**：
   勾選「自備款比例 (%)」，並輸入成數（例如：`20` 代表 20% 成數）。
   勾選「自備款金額 (元)」，直接輸入手頭現有的自備款金額。
3. **輸入貸款條件**：輸入欲試算的「貸款利率」（年利率，如 `2.15`），並透過下拉選單選取「貸款年限」與「寬限期」。
4. **執行計算**：點擊 **[開始計算]** 按鈕，右側將顯示貸款總額、每月應繳、首期本息、總利息與總還款額。
5. **重置重設**：點擊 **[清除重設]** 按鈕，即可快速清空當前欄位，開啟下一次試算。
